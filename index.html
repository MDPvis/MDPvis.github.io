MDPvis: Connect to a Simulator

Wildfire Simulator

Connect to our wildfire suppression simulator and optimizer. In this domain, rewards are given by tree harvest volumes and costs are incurred by spending money on wildfire suppression. Letting a wildfire burn may reduce the harvestable timber, but it will also reduce fuels on the landscape and fire suppression expenses. This is a very challenging large state MDP.

Since the wildfire simulator is computationally too expensive to evaluate for all the possible parameter combinations, we implemented a Model Free Monte Carlo with independencies (MFMCi) algorithm, which builds trajectories from a database of state transitions. You can read more about MFMCi at its repository.

Run Your Own Simulator

If you start a server running on your computer that supports CORS, you can connect this visualization to your domain without hosting the visualization code. The server examples we include in MDPvis's code base include CORS by default.

Add Your Simulator to this List

If you make a server publicly accessible, please let us know so we can list it here. If you need help integrating your server, please open an issue so we can help. We have documentation in the README.

About

This visualization allows you to explore a Markov Decision Process (MDP). The controls provide means of exploring a changing reward, transition, and policy functions towards discovering bugs and understanding the system.
